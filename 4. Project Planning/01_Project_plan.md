# Project Plan

## Project Title

Food Ordering Behaviour and Consumer Trends: A Structured Analysis of Choices and Habits

---

# Project Objective

Develop an interactive Tableau dashboard that analyzes food ordering behavior, customer preferences, revenue, and delivery performance to provide meaningful business insights.

---

# Project Scope

The project includes:

- Data Collection
- Data Cleaning
- Data Analysis
- Dashboard Development
- Interactive Filters
- KPI Design
- Business Insights
- Documentation
- GitHub Repository

---

# Development Methodology

The project follows a phased approach:

1. Requirement Analysis
2. Data Preparation
3. Dashboard Design
4. Dashboard Development
5. Testing
6. Documentation
7. Deployment to GitHub

---

# Deliverables

- Clean Dataset
- Tableau Workbook
- Interactive Dashboard
- Business Insights
- Documentation
- GitHub Repository
