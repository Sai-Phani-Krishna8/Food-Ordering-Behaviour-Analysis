# Proposed Solution

## Solution Overview

The proposed solution is an interactive Business Intelligence dashboard developed using Tableau. It analyzes food ordering behaviour and consumer trends using historical transaction data.

The dashboard enables users to monitor key business metrics through interactive visualizations and filters.

---

## Features

- KPI Cards
    - Total Orders
    - Total Revenue
    - Average Rating
    - Total Delivery Fee

- Interactive Filters
    - City
    - Cuisine
    - Meal Type

- Business Visualizations
    - City-wise Orders
    - Cuisine Popularity
    - Revenue by Meal Type
    - Age Group Analysis
    - Delivery Time Distribution
    - Restaurant Type Analysis

---

- Easy business monitoring
- Faster report generation
- Better customer understanding

- Improved operational planning
