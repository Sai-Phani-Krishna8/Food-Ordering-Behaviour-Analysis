# Non-Functional Requirements

## Performance

- Dashboard should load within a few seconds for the provided dataset.

---

## Usability

- Easy-to-understand layout
- Interactive filters
- Clear visualization labels

---

## Reliability

- Data displayed should accurately represent the dataset.

---

## Scalability

- Dashboard should support additional records without redesigning the layout.

---

## Maintainability

- New KPIs and charts should be easy to add.

---

## Compatibility

- Compatible with Tableau Public/Desktop.
- Dataset stored in CSV format.

---

## Security

- No sensitive personal information is stored.
- Read-only analytics dashboard.
